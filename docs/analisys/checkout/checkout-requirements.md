# Checkout Module Analisys

## Module

Checkout

---

## Objective

Allow customers to complete the purchase process after selecting products.

---

## Functional Requirements

### FR-CHK-001

Users can access checkout.

---

### FR-CHK-002

Users can review delivery address.

---

### FR-CHK-003

Users can review billing information.

---

### FR-CHK-004

Users can review products before payment.

---

### FR-CHK-005

Users can place an order.

---

### FR-CHK-006

Users receive an order confirmation.
