# Cart Test Plan

## Objetive

Validate that users can add product to the cart and manage it before checkout.

---

## Scope

Included

- Add Product
- View Cart
- Remove Product
- Quantity Update
- Continue Shopping
- Checkout

---

Excluded

- Payments
- Orders

---

## Test Levels

- UI
- Functional
- End-to-End

---

## Test Types

- Smoke
- Regression
- Positive
- Negative

---

## Entry Criteria

Application available.

Products page accessible.

User logged in.

---

## Exit Criteria

Smoke Passed.

Regression Passed.

No Critical Defects.
