# Checkout Test Plan

## Objetive

Validate that users can complete the purchase process after selecting products.

---

## Scope

Included

- Checkout
- Review Order
- Address Validation
- Place Order
- Confirmation

---

Excluded

- External Payment Gateway

---

## Test Levels

- UI
- Functional
- End-to-End

---

## Test Types

- Smoke
- Regression
- Positive
- Negative

---

## Entry Criteria

Application available.

Products page accessible.

User logged in.

Products added to cart.

---

## Exit Criteria

Smoke Passed.

Regression Passed.

No Critical Defects.
