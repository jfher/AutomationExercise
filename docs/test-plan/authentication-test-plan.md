# Authentication Test Plan

## Objective

Validate that users can authenticate securely and correctly.

---

## Scope

Included

- Login
- Logout
- Register
- Delete Account

Excluded

- Password Recovery
- MFA
- Social Login

---

## Test Levels

- Functional
- UI
- End-to-End

---

## Test Types

- Smoke
- Regression
- Negative
- Positive

---

## Entry Criteria

Application deployed.

Environment available.

Existing user available.

---

## Exit Criteria

All Critical tests Passed.

No Critical defects.

Smoke Passed.

Regression Passed.
