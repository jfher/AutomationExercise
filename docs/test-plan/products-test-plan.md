# Products Test Plan

## Objective

Validate that users can browse and inspect products correctly.

---

## Scope

Included

- Product listing
- Product search
- Product details
- Categories
- Brands

Excluded

- Shopping Cart
- Checkout
- Payments

---

## Test Levels

- UI
- Functional
- End-to-End

---

## Test Types

- Smoke
- Regression
- Positive
- Negative

---

## Entry Criteria

Application available.

Products page accessible.

---

## Exit Criteria

Smoke Passed.

Regression Passed.

No Critical Defects.
